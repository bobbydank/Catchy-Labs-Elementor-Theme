## How to install

Well, here's how I do it:

```
cd wp-content/themes
```
```
wget https://files.catchylabs.com/themes/catchylabs-theme/catchylabs-theme.zip
```
```
tar -xvf catchylabs-theme.zip
```
```
cd catchylabs-theme/admin
```
```
composer update
```
